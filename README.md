# Priority Inbox Notification System

A simple web application built using **Python (Flask), HTML, CSS, and JavaScript** that displays the **Top 10 unread notifications** based on priority and recency.

## Features

- Top 10 unread notifications
- Priority-based sorting
- Recency-based ordering
- Simple and responsive UI

## Tech Stack

- Python
- Flask
- HTML
- CSS
- JavaScript

## Run the Project

```bash
pip install -r requirements.txt
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
